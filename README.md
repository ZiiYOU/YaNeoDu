# 주요 기능
- API를 통해 자격증 시험 일정 및 합격 후기 조회
- 게시판을 통해 합격자 후기 및 관련 Q&A 커뮤니티 (CRUD)
- 본인이 취득한 자격증 등록 & 다른 유저에게 공유 가능한 프로필 페이지
- 로그인 및 회원가입을 통한 인증/인가
- 별도의 관리자 페이지에서 자격증 진위 확인
